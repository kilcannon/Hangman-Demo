# Hangman-Demo

https://kilcannon.github.io/Hangman-Demo/

Welcome to my Hang Man project!

This app gives you the opportunity to play Hang Man leveraging a small library of pre-existing words.

Technology includes:
+ HTML
+ CSS
+ Javascript
+ JQuery

Theme: Fantasy & Magic

Current features include:
+ Dynamic updating of chosen letters (correct & incorrect)
+ Ability to start over with button reset
+ Counters to track wins/losses

Features to come:
+ Game details persist after page refresh
+ Update to allow to people to play against each other from different terminals
+ Cosmetic upgrades on user interface

Feedback is always appreciated!
